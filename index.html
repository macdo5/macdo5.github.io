Database Administration

In 2018 I was employed as a Database Administrator (DBA), monitoring Microsoft SQL Servers deployed in New Zealand and Australia. My job is to work with a team of DBAs and learn how to use tools to monitor, diagnose and repair SQL Server instances.

Linux

Linux is a very popular operating system for servers because of its compatability with the LAMP bundle, its advanced security and its networking capabilities.

Otago Polytechnic has courses in Linux and Networking, and both of these papers were instrumental in securing my first internship at Iris Data Science in Dunedin.

I was given the opportunity to exercise my problem-solving skills and to develop my knowlege of Linux even further while working at Iris Data Science with great success.

In my current DBA role I am very excited to work with Microsoft SQL Server 2017, which is now available on Linux and Docker as well as Windows. Since 1989 SQL Server has been available for Windows operating systems which has limited the deployment options for developers and database administrators. Combining my knowledge of Linux and SQL Server is a great way to improve as a DBA and to introduce new technologies to my team.
